## OS/8
- 下記のdisk imageで起動できました．まだいろいろ怪しいです．
- [CPLDP-8 by omiokone](https://omiokone.sakura.ne.jp/cpldp8/index.htm) cpld83C.zip中にある os8.rk05
- [pdp8online:PDP-8 File list for os8](https://www.pdp8online.com/images/images/os8.shtml) にある，advent.rk05 と diag-games-kermit.rk05
```
dd if=os8.rk05               of=sd.dsk
dd if=diag-games-kermit.rk05 of=sd.dsk bs=512 seek=8192  conv=notrunc
dd if=advent.rk05            of=sd.dsk bs=512 seek=16384 conv=notrunc
```
のようにして作ったsd.dskをddでSDメモリの先頭ブロックから書き込みます．
/dev/xxx は/dev/sdbなど，生のsdメモリの場所です．ちゃんと調べて間違えないように書き込んで下さい．間違えるとPCのディスクを破壊する可能性があります．
```
dd if=sd.dsk of=/dev/xxx
```

### 起動方法
- SDメモリをTangConsoleのスロットにセットします．
- コントロールパネルのUniversal Monitorを起動します．
- SW1で複数のイメージから起動ディスクを選択できますが，暫定的な措置なので詳細は省略します．
